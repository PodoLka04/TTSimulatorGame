# Тунг-тунг Тик ток Симулятор

Проект создан для публикации на платформе **Yandex Games** с целью коммерции.

**Платформа:** WebGL (для ПК и мобильных устройств)

## 🎮 О проекте

**Тунг-тунг Тик ток Симулятор** — это игра-симулятор TikTok с итальянскими мемами. Листай культовые видео: Тунг-тунг саур, Тралалеро тралала, Роналдо и другие угарные ролики. Забудь про обычный тикток — тут только итальянский брейнрот!

## 🎯 Основные механики

- **Листай видосы** — вверх/вниз, как в привычных приложениях.
- **Итальянские мемы** — Роналдо, Тунг-тунг саур, Тралалеро и другие.
- **Бесконечный брейнрот** — остановиться невозможно.

## 🚀 Сборка и запуск

1. Скачайте репозиторий.
2. Откройте проект в Unity (версия из `Game/ProjectSettings/ProjectVersion.txt`).
3. Для запуска в редакторе откройте сцену `Game/Assets/Scenes/MainScene.unity` и нажмите `Play`.
4. Для публикации на Yandex Games соберите проект в **WebGL** через `File → Build Settings`.
5. Загрузите сборку в личном кабинете разработчика Yandex Games.

The project was created for the **Yandex Games** platform in profit.

## 🎮 About the project

**Tung-Tung TikTok Simulator** is a TikTok simulator game featuring Italian memes. Scroll through iconic videos: Tung-Tung Sahur, Tralalero Tralala, Ronaldo, and other hilarious clips. Forget about regular TikTok — this is pure Italian brainrot!

## 🎯 Core mechanics

- **Scroll videos** — up/down, just like in real apps.
- **Italian memes** — Ronaldo, Tung-Tung Sahur, Tralalero, and more.
- **Endless brainrot** — you won't be able to stop.
